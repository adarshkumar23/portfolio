# Adarsh Kumar — Portfolio

Midnight Ocean Cyberpunk portfolio built with React + Vite.

## Deploy

### Option 1: Vercel (Recommended)
```bash
npm i -g vercel
vercel --prod
```

### Option 2: Netlify
```bash
npm run build
# Upload `dist/` folder to Netlify
```

### Option 3: GitHub Pages
```bash
npm run build
# Push dist/ contents to gh-pages branch
```

## Local Dev
```bash
npm install
npm run dev
```
